## Updater.py v3.1.0 (2026-1-12) - 私人倉庫支援更新
這個版本主要升級了更新器，現在可以支援私有的 GitHub 儲存庫了。

### ✨ 新增功能
- [更新器(Updater.py)] 新增「設定 Token」功能，現在支援更新私人 GitHub 倉庫的檔案。
- [更新器(Updater.py)] 新增設定檔 `updater_config.json` 來儲存 Token (請勿上傳此檔)。

### 🛠️ 優化與變更
- [更新器(Updater.py)] 優化使用者體驗：移除檢查更新失敗時的彈出視窗，改為在介面上顯示紅字提示。
- [更新器(Updater.py)] 修正 UI 排版，將按鈕改為 outline 風格。
- 修改 .gitignore，確保設定檔不會被上傳到 Git。
- 完善了 create_manifest.py，自動排除敏感設定檔。

### 🐛 錯誤修復
- 修復了在沒有 Token 的情況下，存取私人倉庫會直接報錯崩潰的問題。
